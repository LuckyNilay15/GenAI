Langchain models:

It is designed to facilitate interaction with various language and embedding models.
It provides a common interface for different models.

Models---->Language Models----> Text input, text output---> chatbots, agents etc
     ----->Embedding Models----> Text input, vector output(embeddings)(representation of text in numerical form/contextual representation)---> semantic search----->RAG Models.

Plan of Action:

Language Models--->closed source---> openAI, claude, gemini
                 --->open source---> huggingface, ollama, etc.

Embedding Models--->closed source---> openAI
                  --->open source---> huggingface etc.

               ---> chat Models---> specialised for conversation---> input(sequence of messages)---> output(sequence of messages).
