Prompts are input instruction or queries given to a model to guide its output.

Prompts---> text based prompts
        ---> multi modal prompts---> image, video, audio, text etc.

Static vs Dynamic prompts:

Static---> fixed prompt, no change in prompt.
Dynamic---> prompt changes based on the input.

Static Prompt:

Now with the static prompts all the control goes to user and if they write something wrong then it will give wrong output altogether.

Now let suppose we are building a research tool so the output should be factual and consistent. So we need to provide a prompt which is factual and consistent. 
and it give output which is research based.

Now let suppose we are building a creative writing tool so the output should be creative and consistent. So we need to provide a prompt which is creative and consistent. 
and it give output which is creative.

So, for this consistency we need to have a template.

Dynamic Prompt:

Here we have a template and we fill the template with the user input selections.

Example:
we have 3 dropdowns:
and user select the qustion, the style and the length
and we pass down these to our template
i.e. what, how and length
so the output will be based on the dynamic input selections and this is known as dynamic prompt.

Chat Prompt Template: dynamic messsages

SystemMessage: you are a helpful {domain} expert.

HumanMessage: explain about {topic}.

PromptTemplate and ChatPromptTemplate in LangChain are tools for creating dynamic prompts.
PromptTemplate handles plain text, ideal for simple, single-turn tasks. ChatPromptTemplate 
structures multi-turn conversations using roles like "System," "Human," and "AI," allowing 
structured, context-aware interactions necessary for chatbots.

----------------------------------------------------------------

Message Placeholder:

A messageplaceholder is a special placeholder used inside a chatpromptTemplate to dynamically insert messages or a list of messages at runtime.
